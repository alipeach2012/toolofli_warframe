# toolofli_warframe

这是个MATLAB的小程序，就是用来统计我的warframe中 prime 部件情况的。在仓库界面按名称排序之后，依次将数量填入'仓库.xlsx'（不按顺序也行，但是方便你下次再统计），然后运行代码，会自动生成整理的表格'情况统计.xlsx'，让你知道你有啥和缺啥，如果出现模版中没有的部件会在代码中提示，修改模版文件‘模版.xlsx’即可。
